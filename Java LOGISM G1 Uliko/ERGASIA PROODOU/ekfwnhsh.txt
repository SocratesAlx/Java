Για τους πίνακες που θα δημιουργηθούν από την βάση που θα φτιάξετε με τα πεδία που δίνονται στις ΒΔ, (πεδία,πίνακες,κανονικοποίηση κλπ) θα πρέπει να φτιάξετε φόρμες οι οποίες θα είναι άρτιες, και θα κρέμονται από συγκεκριμένο δέντρο (μενού) ώστε να γίνεται εισαγωγή, διαγραφή, και ενημέρωση.



Για να μπει κανείς στην εφαρμογή θα πρέπει να γίνεται διαδικασία login.
