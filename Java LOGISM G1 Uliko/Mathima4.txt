GUI (Graphics User Interface)
Στην Java υπάρχουν κάποιες συγεκριμένες βιβλιοθήκες οι οποίες μπορούν να χρησιμοποιήσουν το γραφικό περιβάλλον που μας προσφέρουν τα windows. Αυτές λειτουργούν για την χρήση φορμών (forms applications) αλλά και απλών message boxes. H βασική βιβλιοθήκη για τα τελευταία, είναι η βιβλιοθήκη javax.

JOptionPane.showMessageDialog(parentComponent, args); 
Εμφανίζει ένα messagebox στην οθόνη. To parentcomponent σημαίνει σε ποιό σημείο του δέντρου από προηγούμενες φόρμες αυτο το messagebox βρίσκεται. Ένα messagebox, μπορεί να κρέμεται ή να εξαρτάται από κάποιες άλλες φόρμες. Στην περίπτωση που αυτό δεν συμβαίνει, γράφουμε null.
To args, είναι το μήνυμα που θέλουμε να εμφανίσει, και είναι τύπου string.

Input από κάποιο messagebox: Αναγκαστικά, αντίθετα με τα Scanner objects, τα message boxes, ΠΑΝΤΑ δέχονται ένα text field, δηλαδή ένα string. Για να πάρουμε κάτι το οποίο έχει άλλου είδους τιμή, δηλαδή δεν είναι string, θα πρέπει να πάρουμε το string, και αντιστοίχως να το μετατρέψουμε.

JOptionPane.showInputDiaolog(args)
Η εντολή αυτή μπορεί να χρησιμοποιηθεί σαν μέρος ανάθεσης (δεξί μέρος) και επιστρέφει μία αξία string. Εναλλακτικά, μπορούμε να την χρησιμοποιήσουμε σαν μία κανονική αξία, και να μπει σαν παράμετρος μίας άλλης method.

Data type classes. Κάθε Primitive Data Type έχει και μία αντίστοιχη class η οποία έχει το ίδιο όνομα με τον αντίστοιχο τύπο, αλλά το όνομα αυτό γράφεται με κεφαλαίο. Αυτές οι classes, χρησιμοποιούνται συνήθως για την επεξεργασία των primitive data. 
Π.χ. int (primitive data type) Integer (int class)
     double (primitive data type) Double (double class) κλπ.

Έτσι, για να μετατρέψουμε ένα string σε ένα αντίστοιχο primitive data type, χρησιμοποιούμε την Method Parse από το αντίστοιχο data type class.

π.χ. Integer.ParseInt(string)

Ternary Operator:
Στην Java όλοι οι operators έχουν ένα ή δύο μέλη εκτός από τον μόνο operator που έχει τρία μέλη, και αυτός είναι ο operator (?:) Αυτός χρησιμοποιείται όταν θέλουμε να αποδώσουμε μία ή μία άλλη τιμή δια μέσου μίας boolean operation. Η σύνταξή του είναι η εξής:
<μεταβλητή> = (<λογική πράξη>)?<τιμή 1 (if true) >: <τιμή 2 (if false)>
