ENCAPSULATION:
Στον προγραμματισμό υπάρχουν δύο σχολές. Η μία είναι να χρησιμοποιούμε τα πάντα σαν public, και να είναι απόλυτα προσβάσιμα από όλα τα στοιχεία του προγράμματός μας. Η δεύτερη σχολή υποστηρίζει τον περιορισμό της έκθεσης των members ενός αντικειμένου, κάνοντας τα data private, και την πρόσβαση σε αυτά (για να τα βλέπουμε ή να τα αλλάζουμε) μέσω ενδιαμέσων member functions οι οποίες στην java ονομάζονται setters και getters. Η τεχνική αυτή της απόκρυψης των Data ονομάζεται Encapsulation. 

Setters και Getters. Συνήθως (από σύμβαση) έχουν συγκεκριμένα ονόματα. Οι methods που βάζουν τιμή σε ένα data, έχουν την λέξη set και το όνομα του data (π.χ. setDim1). Είναι πάντα void methods και έχουν σαν παράμετρο μία μεταβλητή ίδιου τύπου με το data στο οποίο θα βάλουν τιμή. Φυσικά μέσα στο σώμα της method γίνεται η ανάθεση, η οποία είναι συνήθως η ίδια με τον constructor. Οι methods που επιστρέφουν την τιμή ενός data , έχουν την λέξη get και το όνομα του data (π.χ. getDim1). Είναι πάντα data type methods, και δεν έχουν παράμετρο, αλλά επιστρέφουν πάντα την τιμή της αντίστοιχης μεταβλητής (είναι του ίδιου τύπου). Στο σώμα τους, έχουν ένα απλό return το data.

CONSTRUCTOR
Ο constructor είναι μία method η οποία είναι υπεύθυνη για την κατασκευή του αντικειμένου και την αρχικοποίηση των data τα οποία περιέχει. Ακόμα και αν εμείς δεν φτιάξουμε ένα constructor, αυτός υπάρχει εξ ορισμού μέσα σε κάθε class που δημιουργούμε. Τις περισσότερες φορές, μπορούμε να κάνουμε ένα constructor, για να ελέγξουμε την δημιουργία ενός αντικειμένου, και τις τιμές που θα παίρνουν τα data. Επίσης, ένας constructor, όπως και όλες οι functions, μπορεί να γίνει και Overloaded.
O constructor είναι μία method που έχει πάντα το ίδιο όνομα με το class. Ο constructor ΔΕΝ ΕΧΕΙ ΤΥΠΟ και ΔΕΝ ΕΙΝΑΙ ΤΥΠΟΥ VOID. O constructor χρησιμοποιείται ΜΟΝΟ κατά την διάρκεια της δημιουργίας του αντικειμένου.
Στην περίπτωση που το όνομα της παραμέτρου του constructor είναι το ίδιο με το όνομα του data, τότε για να τα ξεχωρίσουμε πρέπει να χρησιμοποιήσουμε το this. Ο this δείχνει πάντα στο datar του class, (και γενικότερα στα στοιχεία του class) και όχι στο εσωτερικό στοιχείο της method στην οποία βρίσκεται. Το this συντάσσεται ως ακολούθως: this.όνομα μεταβλητής .
