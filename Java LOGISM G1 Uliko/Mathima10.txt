CLASS Arraylist

Υπάρχει ένα σύνολο από classes που ονομάζεται Collections, το οποίο χρησιμοποιείται για να αποθηκεύσει ένα σύνολο αντικειμένων που έχουν σχέση μεταξύ τους στην μνήμη. Αυτές οι classes δίνουν αρκετές μεθόδους για να οργανώσουν, να αποθηκεύσουν και να πάρουν data χωρίς να χρειάζονται γνώσεις πως τα δεδομένα αποθηκεύονται. 
Το ArrayList<T> (στο πακέτο java.util) μπορεί να αλλάξει δυναμικά το μέγεθός του ωστε να πάρει περισσότερα στοιχεία. Το Τ είναι ένα placeholder, το αντικαθιστούμε με τον τύπο των στοιχείων που θα περιέχει το Arraylist. 
Π.χ.

ArrayList<String> list;

Οι classes με αυτό τον placeholder που μπορούν να χρησιμοποιούν οποιοδήποτε τύπο ονομάζονται generic classes. Μόνο nonprimitive types μπορούν να χρησιμοποιηθούν για να δηλώσουν μεταβλητές και να δημιουργήσουν αντικείμενα από generic classes. Παρ' όλα αυτά, η java, παρέχει ένα μηχανισμό, που ονομάζεται boxing, ο οποίος επιτρέπει στις primitive values να μπουν σαν objects στα generic classes. 

Π.χ.
ArrayList<Integer> integers;

Αυτό, δηλώνει το integers σαν ένα arraylist που περιέχει Integers. Όταν βάλουμε μία μεταβλητή τύπου int μέσα σε ένα ArrayList<Integer>, η αξία αυτή γίνεται boxed (Σαν να τυλίγεται) μέσα σε ένα Integer object. Και όταν παίρνουμε ένα Integer object από μία ArrayList<Integer>, μπορούμε να το αναθέσουμε σε μία μεταβλητή int, και έτσι λέμε πως το αντικείμενο είναι unboxed (unwrapped). 

Συνηθισμένες μέθοδοι του ArrayList:

add 		Προσθέτει ένα στοιχείο στο τέλος της ArrayList.
clear 		Αφαιρεί όλα τα στοιχεία της ArrayList.
contains  	επιστρέφει true αντο ArrayList περιέχει το συγκεκριμένο στοιχείο, αλλιώς επιστρέφει false.
get		Επιστρέφει το στοιχείο του αντίδτοιχου index.
indexOf		Επιστρέφει τον index της πρώτης εμφάνισης του συγκεκριμένου στοιχείου στο ArrayList
remove		(Overloaded) Αφαιρεί την πρώτη εμφάνιση του συγκεκριμένου στοιχείου στο ArrayList
size		Επιστρέφει το αριθμό των στοιχείων μέσα στο ArrayList 
trimToSize	Μειώνει το capacity του ArrayList στο τρέχον αριθμό των στοιχείων.
