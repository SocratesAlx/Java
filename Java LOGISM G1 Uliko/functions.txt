Τί είναι function?
Είναι ένα κομμάτι κώδικα, το οποίο το χρησιμοποιούμε, για να κάνουμε συγκεκριμένες εργασίες που γίνονται πολύ συχνά. 
Οι functions μας δίνουν την δυνατότητα να έχουμε καθαρότερο κώδικα, χωρίς επαναλήψεις.

Είδη function: 
α) void functions. H void function είναι μία function που δεν επιστρέφει αξία, και καλείται σαν μία ξεχωριστή εντολή. (π.χ. printf,scanf,srand,System.out.println)
β) data type functions: Οι datatype functions, είναι functions που επιστρέφουν αξία. Η αξία που επιστρέφουν είναι του τύπου της function, και χρησιμοποιούνται όπου θα χρησιμοποιούσα μία μεταβλητή. (π.χ. rand(),sqr())
