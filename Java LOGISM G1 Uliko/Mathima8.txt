Κάθε array στην Java είναι αντικείμενο. Αύτο σημαίνει πως και ο ίδιος, έχει δικά του data αλλά και methods. Ένα από τα πιο βασικά data που έχει ένας πίνακας είναι το length, το οποίο περιέχει το μέγεθος του πίνακα. Το καλούμε απλά, χωρίς παρενθέσεις (γιατί δεν είναι method) και μας επιστρέφει το μέγεθος.Επειδή ο πίνακας είναι ένα πολύ βασικό container object, δεν έχει πολλές λειτουργίες από μόνος του, και ως γνωστόν έχει σταθερό μέγεθος. 

EXCEPTIONS και EXCEPTION HANDLING:

Ένα exception δείχνει ένα πρόβλημα το οποίο  συμβαίνει καθώς ένα πρόγραμμα εκτελείται. Το πρόβλημα αυτό είναι κατ εξαίρεσην, και γι' αυτό ονομάζεται exception, δηλαδή είναι κάτι σπάνιο. Μία exception, μπορεί να κάνει ένα πρόγραμμα να τερματιστεί, καιέτσι να μην μπορούν να γίνουν κάποια πράγματα τα οποία ίσως είναι σημαντικά. Με το exception handling, μπορούμε να φτιάξουμε fault tolerant προγράμματα, τα οποία μπορούν να κάνουν κάποιες εργασίες ακόμα και αν υπάρξει κάποια exception σε κομμάτια του κώδικα που έχουμε προνοήσει. Συνήθως, βλέπουμε exception handling σε καταστάσεις όπως out of range πινάκων, αδυναμία σύνδεσης με βάση, έλεγχος τύπου μεταβλητής με αντίστοιχη αξία -πχ, να βάλουμε string μέσα σε ακέραιο-. 

Για να διαχειριστούμε μία exception, βάζουμε τον εν λόγω κώδικα σε ένα try statement. To block του try, περιέχει κώδικα που θα μπορούσε να δημιουργήσει ένα exception. To try block το ακολουθεί πάντα ένα catch block το οποίο περιέχει τον κώδικα που διαχειρίζεται το exception αν κάποιο συμβεί. Αν δεν συμβεί, το catch block δεν τρέχει ποτέ. Επειδή οι exceptions μπορεί να είναι διαφόρων τύπων, μπορεί να έχουμε πολλά διαφορετικά catch blocks που διαχειρίζονται κάθε τύπο exception. 


ENHANCED FOR

To Enhanced for διαδράμει μέσα από όλα τα στοιχεία ενός πίνακα χωρίς να χρειάζεται να χρησιμοποιήσει κάποιον μετρητή, και έτσι, αποφεύγοντας την πιθανότητα να βγεί εκτός των ορίων του πίνακα. Η σύνταξη του enhanced for είναι η ακόλουθη:

	for(παράμετρος: όνομα πίνακα)
 		εντολές

Όπου η παράμετρος έχει τον ίδιο τύπο με τα στοιχεία του πίνακα που διαδράμουμε και ένα όνομα και το arrayName είναι το όνομα του πίνακα που διαδράμουμε. Η αξία κάθε στοιχείου του πίνακα, μπαίνει μέσα στην μεταβλητή παράμετρος, και μπορούμε να την χρησιμοποιήσουμε κανονικά. Προσοχή, στο enhanced for, δεν χρησιμοποιούμε για την διαδρομή του πίνακα τον πίνακα όπως στα απλά for με το όνομα[μετρητής] αλλά μέσω της παραμέτρου.
