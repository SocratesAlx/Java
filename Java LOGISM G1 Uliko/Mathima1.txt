H java σαν γλώσσα δημιουργεί ένα compiled project το οποίο μπορεί να τρέξει σε οποιαδήποτε πλατφόρμα. Αντίθετα με τις άλλες γλώσσες που δημιουργούν ένα .exe file, η java χρησιμοποιεί το Java Runtime Environment (JRE) το οποίο βρίσκεται σε οποιαδήποτε πλατφόρμα οποιουδήποτε λειτουργικού, ώστε τρέξει εκεί. Αυτό πρακτικά κάνει την γλώσσα συμβατή με όλες τις πλατφόρμες. 

Java Development Kit (JDK) χρησιμοποιείται για την κατασκευή προγραμμάτων, και περιέχει τον compiler που χρησιμοποιεί κάποιο IDE για να δημιουργήσει ένα πρόγραμμα Java.

Kάθε project έχει ένα Package. Το package είναι ένα container (σαν κουτί) το οποίο περιέχει όλα τα στοιχεία ενός συγκεκριμένου project ή ενός συγκεκριμένου μέρους ενός project. Ένα project είναι δυνατόν να έχει περισσότερα από ένα packages, αλλά αυτά τα packages τα οποία βρίσκονται στο project συνήθως συνεργάζονται μεταξύ τους. Γενικά ένα package μπορεί γίνει imported.
