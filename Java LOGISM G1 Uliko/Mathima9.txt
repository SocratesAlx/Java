Σχηματικά, ένας πίνακας με μία διάσταση, μπορείνα απεικονιστεί σαν μία γραμμή ή μία στήλη με κουτάκια όσα και το μέγεθός του. Το πρώτο κουτάκι είναι το στοιχείο 0, και το τελευταίο είναι το size-1.

Εκτός από τους μονοδιάστατους πίνακες, υπάρχουν και πινακες με περισσότερες από μία διαστάσεις. Ένας πίνακας μπορεί να είναι διδιάστατος, τριδιαστατος κλπ. Ένας διδιαστατος πίνακας σχηματικά απεικονίζεται σαν ένα επίπεδο (τετράγωνο) όπου χωρίζεται σε γραμμές και στήλες, και το πρώτο κουτάκι του (πάνω αριστερά) έχει την τιμή 0,0.
Ένας τριδιάστατος πίνακας απεικονίζεται σαν κύβος ο οποίος χωρίζεται σε μικρότερους κύβους (με το πάνω αριστερά κυβάκι να είναι το στοιχείο 0,0,0)
Για πάνω από τρεις διαστάσεις δεν μπορούμε να έχουμε σχηματικές απεικονίσεις.

Για να δηλώσουμε ένα πίνακα με περισσότερες από μία διαστάσεις, γράφουμε
int a[][]; <--- Διδιάστατος πίνακας 4x5
int b[][][]; <---- Τριδιάστατος πίνακας 4x5x6


Σαν ένας rule of thumb, μπορώ να διαδράμω ένα πίνακα με τόσα for το ένα μέσα στο άλλο όσα και οι διαστάσεις του πίνακα.
Άρα:
Μονιδιάστατος πίνακας : Ένα for.
Διδιάστατος πίνακας: Δύο for το ένα μέσα στο άλλο.
Τριδιάστατος πίνακας: Τρία for το ένα μέσα στο άλλο μέσα στο παράλλο.

Variable Length Argument Lists (VarArgs)

Με τα VarArgs μπορούμε να δημιουργήσουμε μεθόδους οι οποίες δέχονται ένα αδιευκρίνιστο αριθμό από παραμέτρους. Ένας τύπος που ακολουθείται από τρεις τελείες (elipsis - αποσιωπητικά) στην παράμετρο μίας μεθόδου, σημαίνει πως η μέθοδος δέχεται ένα αριθμό από παραμέτρους του συγκεκριμένου τύπου. Τα αποσιωπητικά μπαίνουν μόνο μία φορά στην λίστα παραμέτρων, και θα πρέπει να μπαίνουν στο τέλος της λίστας αυτής. Μπορούμε να χρησιμοποιούμε method overloading, και να περνάμε arrays κανονικά, για να επιτύχουμε το ίδιο αποτέλεσμα, αλλά τα αποσιωπητικά είναι πιο συμπαγή. 

Command Line Arguments

Στη main method υπάρχει σαν παράμετρος ένας πίνακας από strings με το όνομα args. Αυτός ο πίνακας δέχεται παραμέτρους που δίνονται κατά την διάρκεια της κλήσης του προγράμματος. Αυτές οι παράμετροι, μπορεί να είναι παράμετροι ελέγχου, ή εντολές που ξεκλειδώνουν διαφορετικά επιπεδα του προγράμματός μας (π.χ. moderator mode κλπ).

Class Arrays

Η Class Arrays είναι μία class η οποία έχει τις βασικές static methods για τις συνηθισμένες εργασίες που γίνονται σε πίνακες. Αυτές οι methods συμπεριλαμβάνουν το sort για ταξινόμηση, το binarySearch για αναζήτηση σε έναν ταξινομημένο πίνακα, και το equals για να συγκρίνονται δύο πίνακες, και το fill για να μπαίνουν αξίες μέσα στον πίνακα. Αυτές οι μέθοδοι, είναι overloaded, ώστε να καλύπτουν και primitive data, αλλά και άντικείμενα.
