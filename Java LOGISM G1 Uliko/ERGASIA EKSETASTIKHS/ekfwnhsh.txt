Α) Να γεμίσετε με δεδομένα για πάνω από δύο μήνες τον πίνακα με τις παρουσίες και να βγάλετε μισθοδοσία για εργαζόμενους που  να πληρούν τις παρακάτω περιπτώσεις:

      1) Μισθωτός πλήρους απασχόλησης φούλ μήνας

      2) Ωρομίσθιος που έρχεται 3 ώρες για 3 μέρες κάθε βδομάδας (Δ-Τρ-Πεμ)

      3) Μισθωτός πλήρους απασχόλησης με άδεια κανονική 5 ημερών, και άνευ αποδοχών 3 ημερών μέσα στον ίδιο μήνα.

Οι εργαζόμενοι θα πρέπει να είναι class objects και να ανήκουν σε μία arraylist

Τα αποτελέσματα της μισθοδοσίας θα πρέπει να μπουν σε ένα πίνακα στην βάση ο οποίος θα περιέχει τις μισθοδοσίες. Ο πίνακας αυτο΄ς θα πρέπει να έχει τουλάχιστον τα ακόλουθα πεδία:

Κωδικός εργαζόμενου, ημερομηνία μισθοδοσίας, μισθός, ημέρες αδείας (ασθενείας),ημέρες αδείας (κανονική), ημέρες αδείας (άνευ αποδοχών)

Επίσης θα πρέπει να τυπώνεται (εμφανίζεται) μία απόδειξη μισθοδοσίας του εργαζόμενου, η οποία θα πρέπει να έχει τα παρακάτω στοιχεία:

Κωδικός εργαζόμενου, ονοματεπώνυμο, ΑΦΜ, διεύθυνση, ΑΜΚΑ, μισθός, ημέρες αδείας (ασθενείας),ημέρες αδείας (κανονική), ημέρες αδείας (άνευ αποδοχών)

Τέλος, θα πρέπει να υπάρχει μία γενική μισθοδοσία για όλους τους εργαζόμενους (σε jTable τουλάχιστον) που να περιέχει τα στοιχεία της απόδειξης για όλους όμως τους εργαζόμενους.

  ΠΡΟΑΙΡΕΤΙΚΑ να πάρετε το αποτέλεσμα και να εξάγεται σε excel για την γενική μισθοδοσία, ενώ σε word για την απόδειξη του εργαζόμενου.

Όλα τα queries για την μισθοδοσία ΜΟΝΟ, να γραφτούν σε stored procedures.

Σημείωση: Ένας μισθωτός πλήρους απασχόλησης για κάθε μέρα που δεν δουλεύει χάνει το 1/25 του μισθού του αυτό ισχύει μόνο στις άδειες άνευ αποδοχών.

Ένας ωρομίσθιος παίρνει μέσα στον μήνα το γινόμενο των ωρών επί το ωρομίσθιο.
