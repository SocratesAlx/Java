Extends:
Η εντολή extends δηλώνει πως η class η οποία την χρησιμοποιεί είναι ένας ενισχυμένος τύπος της class την οποία κάνει extend. Εν ολίγοις το extend δηλώνει το inheritance, όπου έχουμε.
ChildClass extends ParentClass. 
Το inheritance δίνει τα data, τις methods της πατρικής class.

Η method paintcomponent
H paintcomponent method, είναι μία μέθοδος που το σύστημα καλεί αυτόματαόποτε θέλει να εμφανίσει ένα DrawPanel. Πρέπει να δηλώνεται για να καλείται κάθε φορά που το Pane εμφανίζεται για πρώτη φορά στην οθόνη, ή όταν είναι καλυμμένη ή αποκαλύπτεται από ένα άλλο παράθυρο, ή όταν το παράθυρο στο οποίο ανήκει αλλάζει διαστάσεις. Η paintcomponent θέλει ένα Graphics object σαν παράμετρο, που αυτό χρησιμοποιείται για να σχεδιάζουμε γραμμές, παραλληλόγραμμα, κύκλους κλπ. Η πρώτη εντολή κάθε paintcomponent method θα πρέπει να είναι πάντα 
super.paintComponent(g);
η οποία κάνει το Panel να είναι σωστά ρενταρισμένο πριν ξεκινήσουμε να σχεδιάζουμε απάνω του. 


getHeight & getWidth

Είναι μέθοδοι που βρίσκονται στην JPanel class οι οποίες επιστρέφουν το ύψος και το πλάτος του JPanel στο οποίο αναφερόμαστε.
(μην ξεχνάμε πως μιλάμε για ένα object το οποίο θα κάνουμε στο κυρίως πρόγραμμά μας substanciate).


Drawline
Είναι μία method η οποία ανήκει στο graphic object g το οποίο έχουμε δημιουργήσει, και τραβάει μία γραμμή σύμφωνα με τις παραμέτρους που βάζω. Οι παράμετροι, είναι οι συντεταγμένες αρχής-τέλους της γραμμής με (width,height) - (width,height)
